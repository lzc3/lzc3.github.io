选项组 参数 作用
操作模式 -c 创建新的归档文件（create）。
-x 从归档文件中提取内容（extract）。
-t 列出归档文件的内容（table of contents）。
文件格式 -z 通过 gzip 压缩 / 解压缩（对应 .tar.gz 或 .tgz 文件）。
-j 通过 bzip2 压缩 / 解压缩（对应 .tar.bz2 文件）。
-J 通过 xz 压缩 / 解压缩（对应 .tar.xz 文件）。
文件操作 -f 指定归档文件的名称（file），必须紧跟文件名（如 -f archive.tar）。
-v 显示详细的操作过程（verbose）。
-C 提取时指定目标目录（Change directory）。
高级选项 -p 保留文件的权限和属性（preserve permissions）。
--exclude 排除特定文件或目录（如 --exclude=logs）。
-P 保留绝对路径（Preserve absolute path），可能导致安全风险。
